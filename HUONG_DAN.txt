## 🚀 Cách sử dụng

### 1. Khởi tạo database
```bash
# Import database
mysql -u root -p --port=3307 < database.sql

# Hoặc tạo tài khoản mới
php setup_accounts.php
```

### 2. Đăng nhập

#### Admin:
1. Truy cập `http://localhost/Waster/login.php`
2. Đăng nhập với: `admin@gmail.com` / `123456`
3. Chuyển đến Admin Dashboard

#### Shipper:
1. Truy cập `http://localhost/Waster/login.php`
2. Đăng nhập với: `shipper@gmail.com` / `123456`
3. Chuyển đến Shipper Dashboard

#### User:
1. Truy cập `http://localhost/Waster/register.php`
2. Đăng ký tài khoản mới
3. Sử dụng các chức năng của người dùng

## 📱 Giao diện theo Vai trò

### User Interface:
- **Trang chủ:** Hiển thị tính năng cho người dùng
- **Bản đồ:** Tìm điểm thu gom bằng GPS
- **Đặt lịch:** Đặt lịch thu gom tại nhà
- **Theo dõi:** Xem timeline xử lý thiết bị
- **Đổi thưởng:** Đổi coins lấy phần thưởng

### Shipper Interface:
- **Dashboard:** Thống kê đơn hàng
- **Đơn chờ:** Xem đơn hàng chờ nhận
- **Đơn của tôi:** Xem và xử lý đơn đã nhận
- **Menu:** Đơn giản, tập trung vào thu gom

### Admin Interface:
- **Dashboard:** Thống kê tổng quan hệ thống
- **Quản lý đơn hàng:** Xem, gán đơn
- **Quản lý người dùng:** User, Shipper, Admin
- **Biểu đồ:** Phân tích dữ liệu
- **Báo cáo:** Xuất báo cáo
